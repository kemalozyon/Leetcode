## 🗂️ Structure

Each topic lives in a numbered folder. Inside, every problem gets its own `.cpp` file containing:

- Multiple solutions ordered from naive to optimal
- Time/space complexity comments per solution
- A `main()` that benchmarks each approach on a sample input

```text
01-ArraysAndHashing/

  ...
```

```bash
g++ -std=c++17 -O2 containsDuplicate.cpp -o containsDuplicate
./containsDuplicate
```
